Tatical Minifier

Quando desenvolvemos aplicações web devemos ter um preocupação em relação ao tamanho dos arquivos que servimos para o usuario esse pequeno script nos ajuda a minificar imagens sem perca de qualidade, ele usa a API do tinyIMG.

Automatizar essa tarefa economizou diversas horas no desenvolvimento de diversos sites, o proximo passo seria integar ele com o Grunt ou Gulp para poder informar o usuario quais imagens de seu projeto ainda não estão minificadas.
